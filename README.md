# First Web Project

This is my first project built completely from scratch using HTML and CSS. The goal of this project is to apply the foundational skills I’ve learned and build a functional and visually structured website.

## Objectives

* Practice HTML structure and semantic elements
* Use CSS for layout and styling (including Flexbox)
* Understand how to organize a simple front-end project

## What I’m Building

A basic website layout that includes sections like a header, navigation bar, and content areas. The focus is on clean structure and layout rather than advanced functionality.

## Skills Demonstrated

* HTML5
* CSS3
* Flexbox layout
* Basic Git and GitHub workflow

## How to View

Clone the repository and open the `index.html` file in your browser.

```
git clone <your-repo-link>
```
